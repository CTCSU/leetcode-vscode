# 概要设计
    Leetcode 只提供了简单、中等、困难的难度区分。题目与题目之间难度差别很大,因此需要客观的分数对题目难度进行打分
# 项目依赖
## 官方vscode插件
- https://github.com/LeetCode-OpenSource/vscode-leetcode/
## 题目难度分
- https://github.com/zerotrac/leetcode_problem_rating/

# 新增内容
    在基础插件上增加了难度分的查询、筛选操作
