```
    v16.17.0
```
### npm版本
```
    8.15.0
```

### 安装node依赖
```
    npm install
```
### 编译ts生成js
```
    tsc
```

### 打包生成vscode插件
#### 安装vsce
```
    sudo npm i vsce -g
```
#### 执行打包
```
    vsce package
```
